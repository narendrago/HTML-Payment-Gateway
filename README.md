# HTML-Payment-Gateway
This is simple HTML donation website (without login) created with Razorpay. This wesbite sends mail to donator once the donation is successfuly completed. This was done as a part of Sparks Foundation-Web Development and Designing Internship-July 2021

## Technologies Used

* HTML
* CSS
* Bootstrap
* Razorpay [Need a pair of testing API keys, can be obtained from their official website-https://razorpay.com/ ]
* 000webhost
